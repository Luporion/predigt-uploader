# Entwicklungsbericht: Release-ZIP v1.5

## Ziel

Eine einfache ZIP-Auslieferung für die lokale Version 1.5 vorbereiten, ohne Vimeo- oder WordPress-Automatisierung und ohne GUI.

## Geänderte Dateien

- `docs/release-v1-5.md`
- `scripts/make-release-zip.ps1`
- `README.md`
- `TASKS.md`
- `STATUS.md`
- `tests/test_manual_test_assets.py`

## Was wurde umgesetzt?

- Release-Anleitung mit ZIP-Inhalt, Ausschlüssen und Entpack-Ablauf für den Gemeinderechner erstellt.
- `scripts/make-release-zip.ps1` erstellt `dist/predigt-uploader-v0.1.5-local.zip`.
- Das Skript nutzt eine Positivliste für die enthaltenen Dateien und entfernt typische Cache-/Arbeitsordner aus dem Staging.
- Dokumentation und Status verweisen auf die neue ZIP-Auslieferung.
- Tests prüfen Release-Skript, wichtige Include-Pfade, Ausschlüsse und Release-Doku.

## Tests

- PowerShell-Syntaxprüfung für `scripts/make-release-zip.ps1`
- `python -m pytest`
- Ergebnis: 107 bestanden

## Offene Punkte / Risiken

- Das Release-Skript wurde syntaktisch geprüft, aber die ZIP wurde in dieser Aufgabe nicht gebaut.
- Externe Programme wie Python, FFmpeg und LosslessCut bleiben weiterhin separate Installationsschritte.

## Nächster sinnvoller Schritt

`scripts/make-release-zip.ps1` ausführen, ZIP auf einem Zielrechner entpacken und dort den Doppelklick-Ablauf testen.
